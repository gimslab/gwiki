# Moniwiki to Markdown 변환 규칙

이 문서는 `*.moniwiki` 파일의 내용을 Markdown으로 변환하기 위한 규칙을 정의합니다.

## 1. 기본 텍스트 서식

*   **굵게:** `'''텍스트'''` -> `**텍스트**`
*   **기울임꼴:** `''텍스트''` -> `*텍스트*`
*   **취소선:** `--텍스트--` -> `~~텍스트~~`

## 2. 제목 (Headings)

`=` 문자의 개수에 따라 제목 수준을 결정합니다.

*   `= 제목 1 =` -> `# 제목 1`
*   `== 제목 2 ==` -> `## 제목 2`
*   `=== 제목 3 ===` -> `### 제목 3`
*   `==== 제목 4 ====` -> `#### 제목 4`
*   `===== 제목 5 =====` -> `##### 제목 5`

## 3. 목록 (Lists)

### 3.1. 순서 없는 목록

`*` 로 시작하는 목록은 Markdown의 순서 없는 목록으로 변환됩니다.

*   ` * 항목 1` -> `* 항목 1`
*   `  * 하위 항목 1.1` -> `  * 하위 항목 1.1`

### 3.2. 순서 있는 목록

`1.`, `a.` 등으로 시작하는 목록은 Markdown의 순서 있는 목록으로 변환됩니다.

*   ` 1. 첫 번째 항목` -> `1. 첫 번째 항목`
*   `  a. 하위 항목` -> `  a. 하위 항목`

## 4. 링크

*   **내부 페이지 링크:** `[페이지 이름]` 또는 `[[페이지 이름]]` -> `[페이지 이름](/pages/페이지 이름.moniwiki)`
*   **외부 링크:** `[[http://example.com|예제 사이트]]` -> `[예제 사이트](http://example.com)`

## 5. 기타

*   **수평선:** `----` (4개 이상) -> `---`
*   **인용문:** `> ` 로 시작하는 줄 -> `> `
*   **코드 블록:** `{{{ ... }}}` -> ````\n ... \n````
*   **메타데이터:** `#`으로 시작하는 줄은 내용 보존을 위해 코드 블록으로 감싸집니다. -> ````\n# ...\n````
