# gwiki 추가 개발 계획 (Phase 2)

이 문서는 gwiki 프로젝트가 초기 버전을 넘어 `prd.md`의 모든 요구사항을 충족하기 위해 필요한 추가 개발 작업을 정의합니다.

---

### 🚀 Phase 4: 요구사항 완비

- [ ] **사용자 인증 시스템 확장:**
  - [ ] `POST /api/auth/register` 엔드포인트 구현 (회원가입 기능)
  - [ ] 사용자 프로필 관리 기능 추가 (예: `GET /api/users/me`, `PUT /api/users/me`)
  - [ ] 프론트엔드에 회원가입 페이지 및 프로필 관리 페이지 UI 구현
  - [ ] 페이지 생성/수정/삭제 시 소유자 기반 권한 확인 로직 추가 (필요시)

- [ ] **검색 기능 강화:**
  - [ ] 백엔드 검색 로직 수정: 페이지 제목과 본문을 모두 검색하도록 개선
  - [ ] 검색 결과에 대한 관련도 순 정렬 알고리즘 적용
  - [ ] 프론트엔드에서 강화된 검색 결과 표시

- [ ] **디자인 및 UX 개선:**
  - [ ] CSS 미디어 쿼리 등을 사용하여 반응형 웹 디자인 강화 (모바일, 태블릿 최적화)
  - [ ] 전반적인 UI/UX 일관성 및 사용성 검토 및 개선

---

### 🧪 Phase 5: 테스트 및 배포

- [ ] **기능 테스트:** 새로 추가된 기능에 대한 단위 및 통합 테스트 작성
- [ ] **사용성 테스트:** 다양한 디바이스에서 UI/UX 테스트 진행
- [ ] **배포:** 최종 버전 빌드 및 배포
